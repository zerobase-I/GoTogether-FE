## 🍀 프로젝트 소개
같이가요는 여행을 함께 하고 싶은 사람들을 찾아 여행을 떠나는 컨셉의 웹앱입니다! 

✅  여행지, 일정, 성별 등 여러 조건을 고려하여 적절한 동행자를 찾아보세요.

✅  실시간 채팅으로 여러 사람들과 소통하며 정보를 공유하고 계획을 만들어보세요.

✅  이벤트 발생 시 알림으로 요청, 수락, 메시지 등 중요한 정보를 놓치지 않고 확인해보세요.

✅  여행이 종료된 후 동행후기를 남겨보세요.
